Use G-Wizard's "Import" function on the Tool Crib to load the custom tool data into the software. You'll need to use it with a particular tool in the crib and select that tool from Feeds and Speeds.

Custom IPT Data Specification

If you're looking to create your own Custom Tooling Data, follow the guidelines in this Excel spreadsheet. Some of the tooling above has links so you can see finished examples.

If you've gone to the trouble of creating Custom Tooling Data, why not contribute it to the community? Just email your CSV file to CNCCookbook. Be sure to include a link to the manufacturer's data you used or a PDF so we can confirm the accuracy of the data for our customers. We'll add it to this list just as soon as we've reviewed it.

Good things will happen for those that contribute!
