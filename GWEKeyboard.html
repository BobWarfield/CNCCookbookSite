G-Wizard Editor: Keyboard Shortcuts

(Note: If you haven't registered for G-Wizard Editor, please do so here!)

 

Because G-Wizard Editor is an editor (no duh!), we designed it with a lot of keyboard shortcuts so you can power your way through editing quickly without constantly reaching for the mouse. Where possible, the keyboard shortcuts are chosen to be mnemonic, and therefore easy to remember. The reason for the choices is given in the "Mnemonic" column to help you remember them.
