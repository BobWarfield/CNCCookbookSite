G-Wizard Editor Install page

Installing GWE on a PC

1. Download the .exe file:

GWE Version 2.71 EXE File (Latest)

GWE Version 2.70 EXE File (Prior Version)

GWE Version 2.33 EXE File

Note: To control where it goes, use right click and use the "Save Link As..." menu choice on the link.

2. Once you get the .exe file onto your machine, double click to install it.

3. Make sure you're registered. You can register right inside G-Wizard once you install it, or you can visit the registration page.

Once you register, you'll get a return email with a confirmation link to click. Once you have confirmed, you're registered.

 

Installing GWE on a Mac

1. Download the .dmg file:

GWE DMG File (Latest)

GWE DMG File (Prior)

2. Once you get the .dmg file onto your machine, right click and select "Open" it to install.

3. It will complain about an "unidentified" developer a little bit, but ignore that. It just means we're not in the App Store.

4. Make sure you're registered. You can register right inside G-Wizard once you install it, or you can visit the registration page.

Once you register, you'll get a return email with a confirmation link to click. Once you have confirmed, you're registered.

 




If you hit a problem, someone else has probably hit the same problem. We've put all the solutions to those problems on the page the "Help Installing" button takes you to.

If you're curious, you can find out what's in the latest release from the Release Notes and Change Log.

 

(Click here if you were looking for G-Wizard Calculator install. Click here if you need to register for G-Wizard Editor.)

G-Wizard Editor runs on Windows and Macs since they both support Adobe Flash and Air.

After installing, the first thing you need to do is click the "Login" button at the top and give it your email. If you're not yet registered, it won't let you do much until you get registered. You will also need to accept the end user license agreement (EULA) to enable the other functions. Lastly, registration is not complete unless you've responded to the opt-in request on the email we send right after registration. That opt-in registers you for our newsletter and it also registers for the beta, so don't ignore it!

If you're having problems, try the "Help Installing" button above.

As always, don't be shy. We want to hear any feedback you may have to help us improve G-Wizard Editor. If you're not already on our mailing list, you'll also be receiving an opt-in email so you can be added to the Beta Test mailing list as well as the G-Wizard newsletter. You will need to be on the mailing list to participate in the Beta test, so don't forget to click the opt-in!
